Glory of the Garden


We are given a JPEG image called garden.jpg:

The file is recognized as a proper JPEG image:

$ file garden.jpg 
garden.jpg: JPEG image data, JFIF standard 1.01, resolution (DPI), density 72x72, segment length 16, baseline, precision 8, 2999x2249, components 3

However, if we show printable strings inside the file contents, we capture the flag (we can specify the length of the string to output):

$ strings -50 garden.jpg
