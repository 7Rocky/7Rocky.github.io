We are given a memory dump called WIN-LQS146OE2S1-20201027-142607.raw:

$ file WIN-LQS146OE2S1-20201027-142607.raw  
WIN-LQS146OE2S1-20201027-142607.raw: data

Reading the name we can assume it comes from a Windows machine.

Flag

The name of the resource looks Base64-encode, and indeed it is the flag:

$ echo SFRCe1cxTmQwd3NfZjByM05zMUNTXzNIP30= | base64 -d  
HTB{W1Nd0ws_f0r3Ns1CS_3H?}
