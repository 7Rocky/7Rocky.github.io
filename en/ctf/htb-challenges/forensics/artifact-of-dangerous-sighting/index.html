Alternate Data Streams

Searching for the name of the DLL file showed that it is a legitimate one, so reverse-engineering the DLL is not required. There is nothing found on the folder that contains the DLL:

The key here is the : notation. In Windows, there is a way to append a file into another file due to NTFS. This feature is called Alternate Data Streams. Notice as well that the challenge acronym is ADS, the same for Alternate Data Streams, so we must be on the right track.
