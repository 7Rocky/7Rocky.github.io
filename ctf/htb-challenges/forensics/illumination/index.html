Illumination

2 minutos de lectura

Tenemos un proyecto de JavaScript que utiliza Git como control de versiones. En el proyecto hay un archivo bot.js y un archivo config.json. En el segundo aparece un texto codificado en Base64:

{
  "token": "Replace me with token when in use! Security Risk!",
  "prefix": "~",
  "lightNum": "1337",
  "username": "UmVkIEhlcnJpbmcsIHJlYWQgdGhlIEpTIGNhcmVmdWxseQ==",  
  "host": "127.0.0.1"
}

$ echo UmVkIEhlcnJpbmcsIHJlYWQgdGhlIEpTIGNhcmVmdWxseQ== | base64 -d  
Red Herring, read the JS carefully

El token parece interesante. A lo mejor podemos encontrar la flag en confirmaciones (commits) antiguos. Con git log podemos ver el histórico de confirmaciones:
