Ingeniería inversa

Lo primero que podemos hacer es descompilar el archivo APK y leer código fuente en Java.

El método s simplemente toma una dirección de correo electrónico de la página inicial. Obsérvese que hay una cadena con una URL: https://anchored.com:4443/test.php, y otro HashMap con una cabecera HTTP:

            public Map<String, String> g() {

Por tanto, podemos deducir que el resultado de uno prp, mrm o frf contendrá la flag. Probablemente, es frf, porque prp podría dar algo como email, y mrm algo como flag ya que se usan como claves del HashMap, no valores (de hecho, si lo miramos, prp devuelve "mail" y mrm devuelve "msg").

Flag

Podemos usar Python para descifrarlo todo y encontrar la flag:

$ python3 -q
>>> from pwn import xor
>>>
>>> ct = b't%u9t8?M/~bx&uz-ebtux8'
>>> key = b'!K!K!KK~K!!KT!KKT!@!KK'  
>>> xor(ct, key)
b'UnTrUst3d_C3rT1f1C4T3s'

Entonces, la flag es:

HTB{UnTrUst3d_C3rT1f1C4T3s}
