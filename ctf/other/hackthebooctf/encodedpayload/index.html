Se nos proporciona un archivo binario llamado encodedpayload:

$ file encodedpayload
encodedpayload: ELF 32-bit LSB executable, Intel 80386, version 1 (SYSV), statically linked, no section header

Ingeniería inversa

Si lo ejecutamos, no aparece nada. Por otro lado, el binario es muy corto:

$ ./encodedpayload
