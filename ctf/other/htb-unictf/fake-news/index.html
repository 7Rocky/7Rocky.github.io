El archivo es tan grande que no lo mostraré aquí. Tiene un documento HTML básico con una etiqueta script que carga un código JavaScript enorme.

Análisis del código JavaScript

Tal vez no sea la forma más segura, pero es la más rápida. Lo que hice fue tomar el archivo PHP y transformarlo en HTML. Luego arranqué un servidor HTTP local y cargué el archivo en el navegador. Este es el resultado:

El código JavaScript se usa para descargar un archivo llamado oficial_invitation.iso automáticamente.

Solo por curiosidad, desofusqué un poco el código y vi que agrega un event listener para detectar el movimiento del ratón. Cuando este evento ocurre, el código crea el oficial_invitation.iso desde bytes en crudo y lo descarga.
